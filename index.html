Taskify

Taskify is a modern and minimalist task management web application built with React. It allows users to create, search, complete, and delete tasks in a simple and intuitive interface focused on productivity.

The application includes real-time task filtering, task completion toggling, and persistent data storage using the browser’s localStorage, ensuring that tasks remain available across sessions.

The UI was designed with a clean layout, soft color palette, and responsive behavior to provide a smooth user experience on both desktop and mobile devices.

My role: I developed the entire application from scratch, handling component architecture, state management with React Hooks, UI styling with CSS, and localStorage integration. This project strengthened my understanding of reusable components, controlled inputs, and application state flow in React.

Zureklamo

Zureklamo is a web application designed to streamline the management of insurance claims between clients and legal professionals. The platform allows clients to submit claims, upload required documentation, track case status, and communicate directly with assigned lawyers.

The system provides lawyers and administrators with tools to analyze cases, request additional information, manage workflows, and oversee the full claim lifecycle from submission to resolution.

Key features include role-based dashboards (client, lawyer, and administrator), real-time messaging, document management, automated email notifications, and a secure, scalable architecture.

My role: I led the full development of the platform, from architecture and UI implementation to backend integration using Next.js, TypeScript, and Firebase (Authentication, Firestore, and Storage). I worked closely with stakeholders to define requirements, iterate on features, and ensure the system supported real-world legal and insurance claim processes end to end.
